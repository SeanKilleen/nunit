**XmlSerializableConstraint** tests whether an object is serializable in XML format.

#### Constructor

```C#
XmlSerializableConstraint()
```

#### Syntax

```C#
Is.XmlSerializable
```

#### Examples of Use

```C#
Assert.That(someObject, Is.XmlSerializable));
```
